# Smart Hire

Smart Hire is an AI-inspired Applicant Tracking System (ATS) designed to streamline recruitment workflows, candidate management, and hiring processes.

## Features

- Candidate Management
  - Track applicants
  - Store candidate information

- Recruitment Pipeline
  - Manage hiring stages
  - Monitor applicant progress

- Job Management
  - Create and manage job openings
  - Organize recruitment activities

- ATS Dashboard
  - View recruitment metrics
  - Track hiring performance

- Database Integration
  - Persistent candidate records
  - Secure data storage using Supabase

## Tech Stack

### Frontend
- React.js
- TypeScript
- Vite

### Backend & Database
- Supabase

- Modern UI Components

## Installation

Clone the repository:

```bash
git clone https://github.com/sabaAnj/Smart_Hire.git
```

Navigate to the project folder:

```bash
cd Smart_Hire
```

Install dependencies:

```bash
npm install
```

Configure environment variables:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

Run the development server:

```bash
npm run dev
```
